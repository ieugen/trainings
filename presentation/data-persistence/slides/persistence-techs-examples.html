Persistence technologies

Some popular Java persistence options

  • JPA - ORM standard (Hibernate, etc.)
  • JDO - Java Data Objects - Object and Relational
  • EJB3 - JPA precursor
  • Other: MyBatis, JOOQ - no O/R mapping
ORM tools are very good if data model is in sync with object model.
Tools like MyBatis may be a good alternative otherwise.
