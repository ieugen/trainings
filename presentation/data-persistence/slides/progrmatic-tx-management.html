Programmatic transactions

Developer manages Transactions via Java code:


    UserTransaction txn = sessionCtx.getUserTransaction();
    txn.begin();
    try {
        TradeOrderDAO dao = new TradeOrderDAO();
        dao.updateTradeOrder(order);
        txn.commit();
    } catch (Exception e) {
        log.fatal(e);
        txn.rollback();
        throw e;
    }
