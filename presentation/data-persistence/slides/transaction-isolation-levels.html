Isolation levels

Levels defined by java.sql.Connection

  • TRANSACTION_NONE
  • TRANSACTION_READ_COMMITTED
  • TRANSACTION_READ_UNCOMMITTED
  • TRANSACTION_REPEATABLE_READ
  • TRANSACTION_SERIALIZABLE

Not all databases(drivers) support all levels !

Isolation is important in concurrent access to DB.

Isolation levels
Isolation is usally implemented with: Locks - read locks - write locks - range locks - detects phantom reads
