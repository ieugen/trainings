Spring Core Container

  • Provides the fundamental parts of the framework (+ DI) via spring-core and spring-beans
  • spring-context provides support for:
    • internationalization via resource bundles
    • event propagation
    • resource loading
    • JMX, remoting and other Java EE features
  • spring-expression enables querying and manipulating the object graph at runtime via a powerful expression language
