Spring Testing framework

  • unit testing and integration-testing with JUnit and TestNG
  • provides consistent loading of ApplicationContext and caching
  • provides mock objects to test code in isolation
  • very powerful in combination with Profiles
Goals: To manage Spring IoC container caching between test execution. To provide Dependency Injection of test fixture instances. To provide transaction management appropriate to integration testing. To supply Spring-specific base classes that assist developers in writing integration tests.
